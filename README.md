Kulamitra – Community Management Web Application

Kulamitra is a modern, role-based community management web application designed to streamline the management of members, events, donations, and community activities within a trust or organization. It provides a centralized platform where Admins can manage operations efficiently, while Members can stay informed and actively participate in community initiatives.

The application features a secure authentication system with support for email/password, Google sign-in, and OTP-based login using Firebase Authentication. Based on user roles, the system ensures restricted and controlled access, where admins have full privileges to manage data, and members have limited, view-based access with selective interaction capabilities.

Kulamitra includes key modules such as Member Management (handling family and locality-based records), Event Management (creating events with RSVP and volunteering features), Announcements (admin-only communication), Gallery (media sharing for community activities), Donations (secure contribution system with category tracking), and Chit Fund Management (transparent handling of community financial schemes).

Additionally, the platform integrates an AI-powered summary module using Genkit, enabling admins to quickly view insights on member statistics, event participation, donation trends, and overall community activity.

Built using a modern tech stack — Next.js, React, TypeScript, Tailwind CSS, ShadCN UI, and Firebase — Kulamitra is designed to be scalable, secure, and user-friendly, making it suitable for managing real-world community ecosystems efficiently.# Kulamitra
