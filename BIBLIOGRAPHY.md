# Bibliography

This document acknowledges the key technologies, libraries, frameworks, and services that were instrumental in the development of the Kulamitra web application prototype.

## Core Framework & Language

-   **Next.js:** The application is built upon the Next.js framework, which provides a robust foundation for server-side rendering, static site generation, and a comprehensive React development environment.
    -   *Citation:* Vercel. (2024). *Next.js by Vercel - The React Framework*. Retrieved from [https://nextjs.org](https://nextjs.org)

-   **React:** The user interface is constructed using React, a JavaScript library for building component-based user interfaces.
    -   *Citation:* Meta and community. (2024). *React*. Retrieved from [https://react.dev](https://react.dev)

-   **TypeScript:** The project is written in TypeScript, a superset of JavaScript that adds static typing for improved code quality and maintainability.
    -   *Citation:* Microsoft. (2024). *TypeScript*. Retrieved from [https://www.typescriptlang.org](https://www.typescriptlang.org)

## UI & Styling

-   **ShadCN UI:** Provides the foundational, accessible, and unstyled UI components that are used throughout the application. It's not a traditional component library but a collection of reusable components.
    -   *Citation:* Vercel. (2024). *shadcn/ui*. Retrieved from [https://ui.shadcn.com](https://ui.shadcn.com)

-   **Tailwind CSS:** A utility-first CSS framework used for all styling, enabling rapid development of a custom design system.
    -   *Citation:* Tailwind Labs. (2024). *Tailwind CSS*. Retrieved from [https://tailwindcss.com](https://tailwindcss.com)

-   **Lucide React:** Provides the clean and consistent icon set used throughout the application.
    -   *Citation:* Lucide Contributors. (2024). *Lucide*. Retrieved from [https://lucide.dev](https://lucide.dev)

## Backend Services & AI

-   **Firebase:** Google's Firebase platform is used for user authentication, providing secure and easy-to-implement login and sign-up flows.
    -   *Citation:* Google. (2024). *Firebase*. Retrieved from [https://firebase.google.com](https://firebase.google.com)

-   **Genkit:** Google's Genkit framework is used for building all AI-powered features, including the community summary generation.
    -   *Citation:* Google. (2024). *Genkit*. Retrieved from [https://firebase.google.com/docs/genkit]

## Key Libraries & Utilities

-   **Zod:** Used for schema declaration and validation, ensuring that data structures (especially for forms and AI flows) are correct.
-   **React Hook Form:** Manages complex form state and validation, working alongside Zod.
-   **date-fns:** A lightweight library used for robust date formatting and manipulation.
-   **recharts:** Used for creating the charts and graphs within the application.

## Placeholder Content

-   **Picsum Photos:** All placeholder images for events, avatars, and the gallery are provided by Picsum Photos.
    -   *Citation:* D. Marby & N. Voigt. (2024). *Picsum Photos*. Retrieved from [https://picsum.photos](https://picsum.photos)
